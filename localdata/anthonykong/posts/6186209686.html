Item 26: Guard against potential ambiguity

Compiler cannot decide: 

void f(int);
void f(char);
 double d = 6.02;
 f(d);                         // error! — ambiguous
Better alternative: explicit cast 

f(static_cast(d));       // fine, calls f(int)
f(static_cast(d));      // fine, cal
