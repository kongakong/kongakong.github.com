Null Pointer in different languages

Each programming language calls it differently: nil, null, none etc… Let’s do a revision.

Objective-c

nil for id. Example: 

NSNumber * num = ...;

if (num == nil) {
    ...
};
NULL for normal C pointer (void *)

Java

null 

Integer num = ...;

if (num == null) {
   ...
}

Visual Basic/VB.Net

Let throw in some VB for fun: Nothing 

Dim num as Integer

If num is Nothing Then
    ....
End If
We can alternatively use function IsNothing to check.

Python

We test for None 

num = ...

if num is None:
   ...

Erlang

There is no concept of Null Pointer in Erlang due to its pattern matching nature.
