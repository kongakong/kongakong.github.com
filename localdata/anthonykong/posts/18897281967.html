How to wrap an exception in a custom exception with original stacktrace and re-raise?

Here is an example: 

import sys
class CustomException(Exception):
    pass

def bad_fun():
    h = None
    h.hi()

try:
    bad_fun()
except Exception, e:
    raise CustomException, e, sys.exc_info()[2]
