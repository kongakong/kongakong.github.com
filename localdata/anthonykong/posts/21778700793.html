I want to enable solarize color scheme in putty. The default color scheme of putty left much to be desired (e.g. blue on black makes it hard to read)

The putty solarize color scheme files can be found here. They are basically two reg files for regedt32 to run and add keys values to putty session definitions.

Alternatively, if you have powershell, you can run the following:

(assuming your the name of the session is ‘server_a’) 

cd HKCU:\Software\SimonTatham\PuTTY\Sessions\server_a
# light schema
Set-ItemProperty -Path .  -Name Colour0  -Value "101,123,131"
Set-ItemProperty -Path .  -Name Colour1  -Value "88,110,117"
