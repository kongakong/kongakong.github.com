It is a follow-up on an earlier post. My solution is 1) Install numpy and scipy via apt-get 

sudo apt-get install python-scipy # this will take care of the numpy dependency
2) create a virtualenv 
 virtualenv  --distribute --system-site-packages  ENV
By using --system-site-packages, python in ENV will be able to use scipy. 3) Starting installing software in the virtual environment 
ENV/bin/pip install pybrain

#python #ubuntu #virtualenv #pybrain
