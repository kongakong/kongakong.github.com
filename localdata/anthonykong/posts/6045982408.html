Item 12: Prefer initialization to assignment in constructors

initialization: 

template<class T>
NamedPtr<T>::NamedPtr(const string& initName, T *initPtr  )
: name(initName), ptr(initPtr)
{}
Assignments in the constructor body: 
 template<class T>
NamedPtr<T>::NamedPtr(const string& initName, T *initPtr)
{
  name = initName;
  ptr = initPtr;
}
Reasons why initialization is better:
  • Make sense for const member variable which can only be initialised
  • More efficient (assignment calls constructor and assignment operator)

Exception:

When there is a large number of data members of built-in types
