Trying to declare a class in objective-c...

Using my fuzzy memory of objective-c syntax, I wrote an objective-c class like this: 
    #import <Foundation/Foundation.h>
    
    @protocol Calculator : NSObject
    
    @end
    
    void handleOperator:(NSString*)oper ;
    void handleNumber:(NSString*)num ;
    void handleReset;

So wrong!

  1. It is @interface not @protocol
  2. return type void should be enclosed in parenthese
  3. @end should be the last line

I wanted to see if I can recall the syntax after all these studies and readings, and I failed miserably. It really takes time to become proficient in another programming language.
