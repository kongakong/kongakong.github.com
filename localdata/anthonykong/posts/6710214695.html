It always take me a while to recall why the java generic is written this way for static. Take this as an example.

It is a common style of static function signature in this module: 
  public static <E extends Enum<E>> ImmutableSet<E> immutableEnumSet(
      Iterable<E> elements) {
We need to specify how the generic type (E) is resolved. Therefore there is an extra ‘<E extends Enum<E> >’ block after public static.
