Item 17: Check for assignment to self in operator=

By pointer, 

C& C::operator=(const C& rhs)
{
  // check for assignment to self
  if (*this == rhs)              // assumes op== exists
    return *this;
   ...
 }
or by reference 
C& C::operator=(const C& rhs)
{
  // check for assignment to self
  if (this == &rhs) return *this;
   ...
 }
