Calculate regular payment based on rate, number of payment and PV

{% codeblock %}
def pmt(i, n, p):
    return i * p * pow((1 + i), n) / (1 - pow((1 + i), n))
{% endcodeblock %}

Example:

{% codeblock %}
>>> pmt(0.06, 120, 120000)
-7206.622591189367
{% endcodeblock %}
